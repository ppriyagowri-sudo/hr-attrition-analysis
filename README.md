# 👨‍💼 HR Analytics Dashboard & Employee Attrition Analysis

## 📌 Project Overview

This project focuses on Employee Attrition Analysis using Python and Power BI. The objective is to identify key factors influencing employee turnover and provide data-driven insights that can help organizations improve employee retention.

The project combines Exploratory Data Analysis (EDA) using Python with interactive Business Intelligence reporting using Power BI.

---

## 🎯 Objectives

- Analyze employee attrition trends
- Identify factors affecting employee retention
- Explore workforce demographics
- Generate actionable HR insights
- Build an interactive Power BI dashboard

---

## 📊 Analysis Performed

### Employee Analysis
- Total Employees
- Attrition Count
- Attrition Rate

### Demographic Analysis
- Gender Distribution
- Age Distribution
- Marital Status Analysis

### Organizational Analysis
- Department-wise Attrition
- Job Role Analysis
- Education Field Analysis

### Financial Analysis
- Monthly Income Analysis
- Salary vs Attrition Trends

### Experience Analysis
- Years at Company
- Years Since Last Promotion
- Work Experience Distribution

---

## 📈 Key Insights

- Certain departments experience significantly higher attrition.
- Employees with lower tenure are more likely to leave.
- Monthly income influences employee retention.
- Job role and work experience impact attrition rates.
- Workforce demographics reveal patterns in employee turnover.

## 🚀 How to Run

### Python Analysis

Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```bash
HR Data Analytics (PYTHON).ipynb
```

Run all cells.

---

### Power BI Dashboard

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the data if required.
3. Explore dashboard insights using filters and visuals.
